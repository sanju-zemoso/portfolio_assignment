GateGuard Pro: Streamlining Campus Exit with React and Firebase

Developed an automated student gate pass system using React, React Native, and Firebase, enhancing campus access management and security. Enabled students to submit requests via QR codes and provided professors with a web interface for authorization. Implemented barcode scanning technology at exit gates for added security.



NFT Music Decentralized App (DApp) Powered by Ethereum Blockchain

Developed a music decentralized application (DApp) on the Ethereum blockchain, empowering users to upload ,buy and sell music assets and play music. Leveraging React JS, I designed the user-friendly front-end interface, while creating smart contracts in solidity on the Ethereum blockchain for secure music ownership and transactions. User Authentication was implemented using the Metamask Wallet. This project demonstrates my proficiency in blockchain technology and decentralized application development.
